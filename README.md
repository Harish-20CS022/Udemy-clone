Udemy Clone Website

This project is a simple **Udemy Clone Website** built using **HTML and CSS**.  
It replicates the basic layout of the Udemy learning platform with a clean and modern design.

Live Demo
https://harish-20cs022.github.io/Udemy-clone/

Features

- Navigation bar
- Course categories section
- Popular courses display
- Clean card-based layout
- Responsive UI design

Technologies Used

- HTML5
- CSS3

Purpose of the Project

This project was built to practice **frontend development skills** by recreating the UI of a popular learning platform (Udemy).  
It helped me improve my **HTML layout structuring and CSS styling skills**.

Project Sections

- Header / Navigation Bar
- Course Categories
- Featured Courses
- Footer Section
